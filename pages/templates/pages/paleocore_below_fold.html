Projects

Paleo Core provides collaborative data management tools for paleoanthropology projects and research teams. Learn about using the paleocore platform to support and facilitate your research.
Tools

Useful software for paleo-data collection, management and collaboration, including data collection tools for field campaigns as well as resources for building and managing geospatial databases.
Community

Standards are key aspects of data exchange. The Paleo Core Project is working with standards organization such as TDWG to develop and document implementations of data standards for paleoanthropology.
Code

Paleo Core is a Free Open Source Software for Geospatial (FOSS4G) project built on Python, Django, PostgreSQL, PostGIS, GeoServer and R. The project source code is available through GitHub.
